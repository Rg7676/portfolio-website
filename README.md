# Rahman Gurbanmuhammedow Portfolio

A minimal, modern, single-page portfolio built with React + TypeScript + Vite.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Features

- Smooth-scroll top navigation
- Dark mode by default + light mode toggle (saved in `localStorage`)
- Hero, Skills, Projects, About, Contact, Footer
- Project cards with expandable details
- Responsive mobile-first layout
- Subtle fade-in animations
- SEO + Open Graph meta tags

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push the repo to GitHub.
2. Import it in [Vercel](https://vercel.com/new).
3. Keep defaults:
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Deploy.

## Update content

Edit portfolio data in `src/data/portfolio.ts`.
